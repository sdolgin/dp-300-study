# DP-300 Study Hub

This repository contains lightweight static study pages for the Microsoft DP-300 exam.

- `index.html` is a High Availability and Disaster Recovery study hub with comparison tables, technology deep dives, scenario quizzes, flashcards, and progress tracking.
- `security.html` is a Security study hub covering encryption, identity, authorization, network security, auditing, compliance, Defender for SQL, Ledger, Row-Level Security, Dynamic Data Masking, and related practice questions.

The site is deployed with GitHub Actions to an Azure Storage static website container. Each page includes a Last Updated footer based on the deployed file's last-modified timestamp.
